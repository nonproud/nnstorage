About NNStorage

เป็น term project ที่ทำในขณะเรียนวิชาหนึ่งเพื่อฝึกการสร้าง cookies โดยใช PHP ซึ่ง NNStorage เป็น CRUD web application ที่พัฒนาด้วย PHP โดยจะแบ่ง User เป็นสอง role คือ Admin และ User ทั่วไป โดย Admin สามารถ Create, Read, Update, และ Delete สินค้าได้ ส่วน User ทั่วไปจะสามารถ เพิ่มสิ้นค้าไปยังรถเข็นและ Checkout ได้ โดยใช้ Cookies
